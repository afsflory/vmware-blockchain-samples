# Pre-requisites
1. docker
2. docker-compose 
3. minikube
4. kubectl 

# Starter kit erc-20 swap steps
1. Install minikube and run the container if it is stopped
2. run ./build-erc20-swap-image.sh 
3. run ./deploy-erc20-swap.sh 
4. get the url from terminal and bring-up UI
